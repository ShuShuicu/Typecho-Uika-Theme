# Uika
> 一款不错的Typecho主题

![Logo](Assets/images/logo.png)

## 快速开始

1.  将主题上传至/usr/themes/文件夹下解压
2.  后台进行启用
3.  访问前台查看效果

### 技术栈
1.  TTDF & Router & TyAjax
>   用于Typecho快速开发
2.  MDUI & ArcoDesign & FontAwesome
>   前端 UI 组件
3.  Vue3 & jQuery & PrismJS & ViewerJS
>   JavaScript 组件
